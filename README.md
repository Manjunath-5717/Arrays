A **comprehensive collection of Java programs** designed for learning and practicing fundamental programming concepts, including **arrays, matrices, loops, methods, recursion, and basic mathematics**. Ideal for beginners, students, and coding interview preparation.

---

## 📂 Programs Overview

### Array Programs
| Program | Description |
|---------|-------------|
| `Arraysum.java` | Calculates the sum of elements in an array. |
| `NoramalArra.java` | Demonstrates basic array operations. |
| `Reverse.java` | Reverses an array. |
| `ScannInputr.java` | Reads array input from the user. |
| `ToreturnAUMethod.java` | Array sum using a method that returns a value. |
| `WThirdArray.java` | Array operations using a third array. |
| `WithUsingThirdA.java` | Demonstrates array usage with a helper array. |
| `WithoutreturnArray.java` | Performs array operations without returning from a method. |

### Matrix Programs
| Program | Description |
|---------|-------------|
| `MatrixMU.java` | Performs matrix multiplication (uppercase variant). |
| `MatrixMULL.java` | Matrix multiplication with a different approach. |
| `Matrixmul.java` | Standard matrix multiplication program. |

### Mathematical Programs
| Program | Description |
|---------|-------------|
| `Factorial.java` | Calculates factorial of a number. |
| `CountNatural.java` | Counts natural numbers up to a given limit. |
| `Nat.java` | Prints natural numbers. |
